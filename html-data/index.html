# Linux

SSH installieren

Ich denke mal jeder der Linux benutzt weiß wie das funktioniert oder hat ssh schon installiert. Hier trotzdem für Ubuntu. 

> sudo apt update 
> sudo apt install openssh-client

Auf die Rechnerhalle verbinden

Nachdem du SSH installiert hast, kannst du dich auf die Rechnerhalle verbinden. Hierbei gibt es zwei Optionen

SSH-Key erstellen

Jetzt erstellen wir ein neues SSH-Key-Pair (public und private). Falls ihr schon einen SSH-Key habt, könnt ihr diesen Schritt überspringen.
Mit der Option -t ed25519 spezifizieren wir,dass die elliptische Kurve 25519 um das Public-Private-Key-Pair zu erstellen. 

> ssh-keygen -t ed25519

SSH-Public-Key auf die Rechnerhalle hochladen

Jetzt musst du deinen Public-SSH-Key an die Rechnerhalle schicken. Erstze hier <rbg-username> mit deinem RBG-Nutzernamen. 

> ssh-copy-id <rbg-username>@lxhalle.in.tum.de
